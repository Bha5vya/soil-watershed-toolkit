# Runoff Accumulation Zones (RAZ)

## Objective
To identify zones where surface runoff naturally accumulates within the Mula–Mutha–Bhima catchment using DEM-derived flow accumulation.

## Concept
Runoff tends to concentrate in valleys and drainage channels. Mapping these zones helps identify suitable locations for water harvesting and flood mitigation structures.

## Tools
Google Earth Engine

## Data
- SRTM DEM

## Output
Runoff accumulation map showing low to high runoff concentration zones.
